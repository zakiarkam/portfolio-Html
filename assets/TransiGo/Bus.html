Introduction

This is the second year group project we did at the University. This project was mentored by London Stock Exchange Group (LSEG)

The TransitGO project aims to revolutionize public transportation in Sri Lanka by addressing key challenges such as lack of real-time information, incident management, and improving the overall passenger experience.

Project Web and Mobile App Pages

Key Features

  • Real-Time Bus Status: Stay updated with the latest bus schedules and delays.
  • Incident Reporting: Report and receive updates on accidents and disruptions.
  • Lost and Found: Manage and retrieve lost items effortlessly.
  • Package Transfer: Secure and track your packages during transit.
  • Community Engagement: Provide feedback and contribute to improving services.
  • Admin Control Panel: Provides administrators with a dashboard to manage the system, monitor operations, and respond to incidents.
  • Role-Based Authorization and Authentication: Implements secure access controls, allowing users to perform functions based on their assigned roles.
